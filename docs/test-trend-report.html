E2E tests are counted via git grep -c "test(" in Bruno *.bru files (V1 + V2 suites) for Feb 2026 onward. Dec 2025 and Jan 2026 use pm.test( in Postman collections (the predecessor format before the Bruno migration in Feb 2026). obj/ build artefacts are excluded from all counts.
